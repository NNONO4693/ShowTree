
./showtree.sh [option] [dossier]

si ne pas specifier le dossier le script executer automatiqment dans le dossier courant

LES OPTIONS
-d pour afficher uniquement les dossiers et les sous dossiers
-s pour une l'affichage simple
-c pour une affichage avec les coloration
-n pour afficher avec la possibilite de specifier le nombre de niveaux visualiser
-nc pour la coloration avec la possibilite de specifier le nombre de niveaux visualiser
-a pour l'affichage avance
